# 프로젝트 작업 가이드 (GEMINI)

## 🛠 CLI 도구 활용 (gws cli)
- **성공 패턴 (Redirect & Read)**: `gws cli`를 사용할 때 결과를 바로 터미널로 출력하지 않고, 파일로 리다이렉션하여 저장한 뒤 읽는 방식이 가장 빠르고 정확합니다.
  - 예시: `gws sheets +read ... > output.json && cat output.json`
- **장점**: 
  - 쉘(Shell)의 인코딩 문제로 인한 한글 깨짐을 완벽하게 방지합니다.
  - 대량의 데이터를 AI가 파싱하기 가장 좋은 깨끗한 상태로 유지합니다.
  - 여러 번의 `help` 조회나 재시도 없이 단 한 번에 성공할 수 있는 검증된 방식입니다.
